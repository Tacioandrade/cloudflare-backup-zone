# Cloudflare Backup Zone

## English

This project backs up every Cloudflare DNS zone that the configured API token can access.

On each run, `backup_cloudflare_zones.py`:

1. Reads a local `.env` file.
2. Lists all Cloudflare zones available to the token.
3. Downloads all DNS records for each zone.
4. Creates a dated backup folder under `backups/` by default.
5. Writes one JSON file per zone, plus `zones.json` and `manifest.json`.
6. Removes older backup folders according to the retention period configured in days.

### Requirements

- Python 3.10.12
- A Cloudflare API token with permission to read zones and DNS records.

### Configuration

Create a local `.env` file based on `.env.example`:

```env
CLOUDFLARE-API-TOKEN=replace-with-your-cloudflare-api-token
BACKUP_RETENTION_DAYS=30
BACKUP_OUTPUT_DIR=backups
```

`CLOUDFLARE_API_TOKEN` is also accepted if you prefer an environment-variable-safe name.

### Usage

```bash
python3 backup_cloudflare_zones.py
```

To use a different environment file:

```bash
python3 backup_cloudflare_zones.py --env-file /path/to/.env
```

Backups are created with UTC timestamps, for example:

```text
backups/2026-06-26_21-30-00/
```

The cleanup step only removes backup directories whose names match the timestamp format generated by the script.

## Português

Este projeto faz backup de todas as zonas DNS da Cloudflare que o token de API configurado consegue acessar.

A cada execução, `backup_cloudflare_zones.py`:

1. Lê um arquivo `.env` local.
2. Lista todas as zonas da Cloudflare disponíveis para o token.
3. Baixa todos os registros DNS de cada zona.
4. Cria uma pasta de backup com data dentro de `backups/` por padrão.
5. Grava um arquivo JSON por zona, além de `zones.json` e `manifest.json`.
6. Remove pastas de backup antigas conforme o período de retenção configurado em dias.

### Requisitos

- Python 3.10.12
- Um token de API da Cloudflare com permissão para ler zonas e registros DNS.

### Configuração

Crie um arquivo `.env` local baseado no `.env.example`:

```env
CLOUDFLARE-API-TOKEN=replace-with-your-cloudflare-api-token
BACKUP_RETENTION_DAYS=30
BACKUP_OUTPUT_DIR=backups
```

`CLOUDFLARE_API_TOKEN` também é aceito caso prefira um nome compatível com variáveis de ambiente.

### Uso

```bash
python3 backup_cloudflare_zones.py
```

Para usar outro arquivo de ambiente:

```bash
python3 backup_cloudflare_zones.py --env-file /path/to/.env
```

Os backups são criados com timestamps em UTC, por exemplo:

```text
backups/2026-06-26_21-30-00/
```

A etapa de limpeza remove apenas diretórios de backup cujo nome siga o formato de timestamp gerado pelo script.
